# State Law Impact: Hawaii SB3202 (Act 39)
## Mandates
- Passed in May 2024, SB3202 requires all Hawaii counties to adopt or amend ordinances to allow **at least two** accessory dwelling units on all residentially zoned lots by December 31, 2026.
- Prohibits private covenants (HOAs) in urban districts from limiting the number of ADUs below what the state allows.
